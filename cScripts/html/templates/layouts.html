var name type = expression
Either the type or the expression part may be omitted, but not both.
If the expression is omitted, the initial value is the zero value for the type, which is zero for numbers, false for booleans, "" for strings, and nil for interfaces and reference types (slice, pointer, map, channel, function).
The zero value of an aggregate type like an array or a struct has the zero value of all of its elements or fields.

var i, j, k int
